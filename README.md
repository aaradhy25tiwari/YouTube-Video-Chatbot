# YouTube-Video-Chatbot
## Requirements for this project are:
+ Python 3.10 and above
+ youtube-transcript-api
+ deepmultilingualpunctuation
+ google-genai
+ python-dotenv

---

## Repository Structure

    YouTube-Video-Chatbot/
    ├── main.py
    ├── requirement.txt
    └── README.md

- **main.py** — the main file which downloades a YouTube video
- **README.md** — this documentation
- **requirements.txt** — List of modules required for this project

---

## Setup Instructions

1. **Clone** the repository:

        git clone https://github.com/aaradhy25tiwari/YouTube-Video-Chatbot.git

        cd YouTube-Video-Chatbot

2. Install dependencies:

       pip install -r requirements.txt

3. Run the web UI:

        python -m streamlit run main.py
        Open http://127.0.0.1:8501 in your browser
